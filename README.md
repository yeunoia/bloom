# @yeunoia/smear

Wrap anything with smear

```tsx
import { Smear } from "@yeunoia/smear"

<Smear>anything</Smear>
```

---

## Install

```bash
npm install @yeunoia/smear
```

## Props

| Prop              | Type                | Default   |
| ----------------- | ------------------- |-----------|
| `backgroundColor` | `string`            | `#A4E7D5` |
| `color`           | `string`            | `inherit` |
| `tip`             | `"flat" \| "round"` | `"round"` |
| `paddingX`        | `number`            | `4`       |
| `paddingY`        | `number`            | `2`       |

`tip="flat"` — highlighter-like edges  
`tip="round"` — brush-like edges

## Requirements

React ≥ 17
